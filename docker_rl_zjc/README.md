# docker-for-RL
##  安装步骤
### 使用dockerfile构建基础镜像

```
git clone https://github.com/ustbmicl-ros2epucksRL/docker-for-RL.git
cd docker_rl_zjc
docker build -t for_rl_image:v1.0 .
```

随后在docker desktop中运行镜像。
vscode连接容器后，可能需要按照提示安装python、jupyter插件。
